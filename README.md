# ArknightsAuthenticationAssistant
明日方舟通行证查询工具小助手，使用Uni-app开发，目前已上线微信小程序版本；明日方舟通行证是明日方舟朝陇山研发的一款周边产品，一般是盲盒亚克力制品
本程序对该产品的分盒、信息、抽取、干员信息可以进行查询
